# Contributing

```text
Welcome to vanyauhalin/fresco sources.

Usage: make <subcommand> [argument=value]
       make install
       make lint project=FrescoCLI

Subcommands:
  analyze     Analyze the project via SwiftLint.
  build       Build the project via Tuist.
  ci          Install only production dependencies via Tuist.
  clean       Clean generated Tuist files except for dependencies.
  dev         Generate a development workspace via Tuist.
  help        Show this message.
  install     Install dependencies via Tuist.
  lint        Lint the project and Tuist directory via SwiftLint.
  test        Test the project via Tuist.
  version     Print the current project version.

Arguments:
  project     Specify a project for the lint command.
```
